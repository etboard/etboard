## ETboard, IoT의 시작 💻

ESP32 기반 Arduino 상위 호환 보드

### 지원 언어
* 아두이노(Arduino)
* 엔트리(Entry)
* 파이썬(MicroPython)

### 교육용 코딩 키트
- 신재생 에너지
- 스마트 화분
- 스마트 팩토리
- 스마트 가로등
- 스마트 모이

### 문서 및 자료(Documentation & File)

아래 자료는 [이티보드(ET-Board) 자료실](http://et.ketri.re.kr/board/)에서도 얻을 수 있습니다.
